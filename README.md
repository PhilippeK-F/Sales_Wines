Sales Wines — Étude de marché Domaine des Croix

Description du projet :

Projet de certification Data Analyst (Wild Code School) — Blocs 2, 3 & 4.
Le client, le Domaine des Croix (Bourgogne, France), souhaite se lancer sur le marché américain du vin. L'objectif est d'analyser un dataset de 130 000 bouteilles pour définir un prix compétitif et convaincre le client avec une analyse de marché complète.

"Comment positionner les vins du Domaine des Croix sur le marché américain de manière compétitive ?"

Le dataset :

- Dataset marché (130k bouteilles — source WildCodeSchool)
- CaractéristiqueDétailNb de lignes~130 000 bouteillesColonnes cléscountry, points, price, variety, province, wineryNotesEntre 80 et 100 (gamme qualitative)Prix filtrés< 200 $ (hors ultra haut de gamme)
- Dataset client (Domaine des Croix)

Analyses réalisées :

1. Analyse du marché global

Répartition du nombre de vins par pays (Top 10)
Pays avec les meilleures notes moyennes (filtre > 500 vins)
Moyenne des notes par cépage (filtre > 700 vins)
Répartition par décile de notes

2. Analyse des prix

Distribution des prix (< 200 $)
Corrélation entre note et prix (scatter plot)
Positionnement prix/qualité

3. Analyse comparative

Comparaison France vs marché mondial
Focus Bourgogne vs autres régions françaises
Focus Pinot Noir bourguignon — concurrents directs

4. Proposition de prix
Recommandation tarifaire pour le Domaine des Croix basée sur l'analyse concurrentielle.

Principaux enseignements :

- Les USA, France et Italie dominent le marché en volume
- La Bourgogne se positionne parmi les régions à la meilleure note moyenne
- Le Pinot Noir est un cépage premium avec des notes élevées
- La note du Domaine des Croix (94/100) le positionne dans le top 10% du marché
- Le prix recommandé se situe entre 40 $ et 70 $ pour être compétitif
